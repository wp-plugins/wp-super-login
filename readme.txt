== Description ==

Wp Super Login plugin, It adds a nice introduction to the theme. 
We will also bring more updates to the availability of 3 languages.
These languages are English, French and Turkish.

== Installation ==

Please follow these steps to install the plugin.

1. Upload `wp-super-login` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Wp Super Login' menu in WordPress
1. Place `<?php if(function_exists('login_add')) { login_add(); } ?>` in your templates
